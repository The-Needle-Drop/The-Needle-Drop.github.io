Lightbulb & Bigger can be better

  • Gursimmer Banwait
  • March 3rd, 2022

Big updates to the project and more changes in bound in this weeks blog post

GOOD NEWS!

The good news, we got the ring light PCB in from JLCPCB. After soldering the LEDs and attaching powering wires we now have the ability to use the light function of our lamp.

BAD NEWS

Unfortunatley, the down side of using cutting edge technology is the growing pains of being an early adopter. Integration of using the new autofocusing camera allows for the end user to place the camera / lamp head in many different angles and potiions while still being able to take a well detailed and perfect picture. The new camera uses a brand new library that is still in it's alpha stages to take pictures. When testing out the ability to take many pictures in a short amount of time, there was a clear memory problem. The MCU doesn't have adequate amount of RAM to store/take pictures and overwrite the previous ones. This issue did not persist when running the same script on the Raspberry Pi 3B+. Rather than spend time to try and get the RPi Zero to work by itself the groupn decided for a minimum viable product with reproducibility and adopt using the RPi 3B+.

Future Work

In the hardware of things this means that we need a brand new case design to hold the battery, the new MCU, and a new way attach to existing lamp neck that has been fit the lamp and such. Futher work needs to be done on the mech side so join the ribbon cable signals to the camera in the lamo head aswell.

For the software side of things, developing the start up script to automatically have the MCU funciton as intended needs to be finalized. As we are now connecting to the internet for file transfer the start up script will have the MCU will boot up, the camera will take a picture and upload to the gdrive.
